# ash: a C-shell

ash is a UNIX-like shell clone in C that reads, parsers, and executes commands similar to popular shells like Bash or Zsh.

## Features

* Builtin commands
* External program execution via PATH lookup

## Future Improvements

* Quoting
* Piping and Redirection
* Command History
